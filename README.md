# Clinic Management System ER Diagram

This repository contains the ER diagram for a simple clinic management system.

## About
The design models how a clinic handles patients, doctors, appointments, consultations, tests, reports, and payments in a structured way.

## Key Features
- Patients can book multiple appointments with doctors  
- Appointments may lead to consultations  
- Doctors can prescribe multiple tests during a consultation  
- Each test generates a report  
- Payments are linked to consultations
